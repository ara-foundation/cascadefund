# CascadeFund
CascadeFund is building ecosystem of open-source maintainers and users.
Landing page is in dialog with the CascadeFund founder.

-   🚀 **App URL:** https://cascadefund.org/

> CascadeFund is part of the Ara project.

## 🛠️ Development

### Requirements

-   **Node.js** (LTS recommended)
-   **pnpm** (required)

Install pnpm:

``` bash
npm install -g pnpm
```

### Setup

``` bash
git clone https://github.com/ara-foundation/cascadefund.git
cd app
pnpm install
pnpm run dev
```

> Don't forget to prepare your .env using `.env.example`
