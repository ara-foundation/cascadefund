## 🛠️ Development

### Requirements

-   **Node.js** (LTS recommended)
-   **pnpm** (required)

Install pnpm:

``` bash
npm install -g pnpm
```

### Setup

``` bash
git clone https://github.com/ara-foundation/cascadefund.git
cd app
pnpm install
pnpm run dev
```

> Don't forget to prepare your .env using `.env.example`
